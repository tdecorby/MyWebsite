My Name is Taylor, and welcome to my

Web Portfolio

In all the websites that I created, I used HTML5, CSS3, jQuery, and JavaScript. Some of the websites have a variety of plug-ins to enhance the website.
NOTE: Some of the websites were not made for tablets or phones.

University of Winnipeg ACS Website - 2015

This is a Project that I did for the University of Winnipeg's Applied Computer Science Division.
This is a dynamic website that was made using PHP and MySQL database.
You can go to the admin page here, the username and password are "demo".
This website had to be dynamic to allow for a non-technical person to be able to manage it without assistance.

ITSS - 2013

I built this website for the ICT Division of the Finance Department of the Government of Manitoba in Summer 2013.
This was an interesting project. It was running on Microsoft Sharepoint so I was heavily modifying the CSS of the Sharepoint website to change the way it looked and acted. Then I realised that they didn't need any of the Sharepoint functionality so I scrapped that and just made this in HTML.
It had to have compatibility with Internet Explorer 7. This made it very difficult to work with but I still think turned out nicely. I hope you agree.

Bakery - 2011

I made this as a sample website for a Bakery.
I used this opportunity to test out some photo slide showers, and make web pages that take up the full pages real estate.
It doesn't have much functionality but it was mostly for design practice.
